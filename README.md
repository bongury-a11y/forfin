# ForFin 가계부

> Personal finance web app — Phase 2 Beta

## 개요
- 다중 통화 지원 가계부 (5개 언어)
- 로컬 저장 (LocalStorage) + 선택적 클라우드 동기화 (Supabase)
- 단일 HTML 파일 (Tailwind CDN + Vanilla JS)

## 기술 스택
- HTML5, Tailwind CSS, Vanilla JavaScript
- Supabase (인증 + 데이터베이스)
- Cloudflare Pages (배포)

## 주요 기능
- 수입/지출/이체 관리
- 다중 자산 + 다중 통화
- 정기지출 자동 처리
- 분석 대시보드 (10개 카드)
- 5개 언어 지원 (ko/en/jp/cn/es)
- 다크/라이트 모드
